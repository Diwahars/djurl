# Dj-URL

An url shortener built using Django.
Uses [Materialize](http://materializecss.com) for the UI.

![Screenshot](http://i.imgur.com/Sr4vtKc.png)

### Install

1. Clone the repo.
2. Install dependencies.
3. cd to the direcoty and run `./manage.py runserver`

### Dependencies

+ Django 1.7.1
